# DoubanMovie
1.就是一个用豆瓣电影提供的接口,然后用Swift2.0写的一个和电影相关的APP,纯为练手.      
2.因为豆瓣为普通开发者提供的接口有限,所以做出来肯定和豆瓣自己的APP有很大差别.      
3.又因为缺少设计师的原因,很多图都没有,后期可能会补上,也希望射鸡湿朋友可以为我供图,为开源做点贡献:)
